## What is Patcher?
Patcher is a Python library **and** CLI for macOS that leverages the Jamf Pro API to fetch patch management data and generate comprehensive reports in varying formats. It simplifies tracking and reporting on software update compliance across macOS devices managed through Jamf Pro.

The CLI (`patcherctl`) is the primary interface for most users. The underlying logic is also importable. See the [library quickstart](https://docs.patcherctl.dev/getting-started/setup/library.html) for using Patcher programmatically from your own scripts or services.

Read the full project documentation [on our project homepage](https://docs.patcherctl.dev).

## Installation

### Using UV (Recommended)
[UV](https://github.com/astral-sh/uv) is a fast Python package installer and resolver. Install UV first:

```console
# On macOS and Linux
$ curl -LsSf https://astral.sh/uv/install.sh | sh

# Or using pip
$ pip install uv
```

Then install Patcher:

```console
$ uv pip install patcherctl
```

### Using pip
Alternatively, you can install using pip:

```console
$ python3 -m pip install patcherctl
```

> [!NOTE]
> Patcher is both a CLI tool and a Python library. Most users will reach for `patcherctl`, but the underlying `patcher` package is importable for custom integrations. See the [library quickstart](https://docs.patcherctl.dev/getting-started/setup/library.html) for the importable surface.

## Usage
For a list of all available commands and options, visit the [Usage section](https://docs.patcherctl.dev/usage/export.html) of our documentation.

After installation, reports can be generated by running the `export` command. You can specify the output directory for the reports and choose to generate PDF reports alongside Excel files.

```console
$ patcherctl export --path '/path/to/output/directory'
```

## Development Setup

For development, we recommend using UV for faster dependency management:

```console
# Clone the repository
$ git clone https://github.com/liquidz00/Patcher.git
$ cd Patcher

# Install UV if not already installed
$ curl -LsSf https://astral.sh/uv/install.sh | sh

# Set up full monorepo dev environment (patcherctl + patcher-api + all extras)
$ make dev

# Run tests
$ make test

# Run linting and formatting
$ make lint
$ make format
```
